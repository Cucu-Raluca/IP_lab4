# 📱 Socially: Your AI-Powered Digital Concierge

**Socially** is an innovative mobile and web application designed to bridge the gap between digital planning and real-world social interaction. By leveraging machine learning and deep calendar integration, Socially eliminates "analysis paralysis," transforming the way groups coordinate and experience life together.

---

## 🚀 Key Features

* **AI-Driven Recommendations:** A personalized engine that learns your "vibe" to suggest tailored events and itineraries.
* **Smart Availability Sync:** Automatically identifies "windows of opportunity" by syncing with **Google Calendar** to find mutual free time among friends.
* **Real-Time Deal Sourcing:** An autonomous **AI Agent** scans the web for the best ticket prices and accommodation deals.
* **Collaborative Planning:** Dynamic group environments where users can vote on activities, comment on segments, and finalize shared schedules.
* **Secure Transactions:** Integrated with high-security payment services for seamless bookings.

---

## 👥 System Actors

| Actor | Role |
| :--- | :--- |
| **Logged User** | The core participant who manages friends, syncs calendars, and initiates group planning. |
| **Unlogged User** | Guests who can browse public landing pages before creating a secure profile. |
| **AI Agent** | The "brain" of the app; processes complex datasets and filters real-time market data. |
| **Admin** | Ensures system stability and manages local event logistics (beverages, meals, etc.). |
| **Payment Service** | External entity handling secure financial handshakes and encryption. |



---

## 🛠️ Technical Stack (Core Logic)

The project utilizes **Java** for its robust backend logic, focusing on Object-Oriented principles to manage the complex interactions between users and the AI Agent.
